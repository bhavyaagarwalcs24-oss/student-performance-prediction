# Student Performance Prediction

A Machine Learning web application built using Python, Streamlit, and Random Forest Regressor.

## Features
- Predict student final marks
- Interactive web interface
- ML-based prediction system

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Random Forest Regressor

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Output

Predicts student marks based on:
- Study Hours
- Attendance
- Previous Marks
- Assignment Score
